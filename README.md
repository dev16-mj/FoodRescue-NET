
# 🍽️ FoodRescueNet

## Smart Surplus Food Redistribution System

FoodRescueNet is a web-based platform that connects restaurants, hotels, event organizers, and households with NGOs and shelters to reduce food wastage.

## 🚀 Run Locally

### Prerequisites:
- Node.js

### Steps:

1. Install dependencies  
   npm install

2. Set the GEMINI_API_KEY in .env.local

3. Run the app  
   npm run dev
